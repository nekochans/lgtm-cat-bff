# lgtm-cat-bff

[LGTMeow](https://lgtmeow.com) の BFF(Backend For Frontend)

# Getting Started

## Package のインストール

```bash
npm ci
```

## ローカル環境の起動

最初に wrangler を使って認証を行います。

```bash
npm run login
```

実行するとブラウザが起動して cloudflare のページに遷移して認証・認可が完了します。

これが完了するとローカル環境の起動や環境変数の追加等が出来るようになります。

以下でローカルサーバーが起動します。

```bash
npm run start
```
